# Sangel - официальный сайт проекта

Этот репозиторий содержит первую версию веб-сайта стартапа **Sangel** - сервиса экстренной помощи и умного украшения (браслета/кулона), который позволяет быстро вызвать ГБР в опасной ситуации.

Сайт представляет компанию, рассказывает о продукте и архитектуре сервиса, а также служит точкой входа для пользователей и партнёров.

## О проекте

Веб-сайт служит:
- посадочной страницей приложения и устройства Sangel  
- источником информации о продукте и функциональности  
- площадкой для презентации компании и команды  
- базовой промо-страницей для первых пользователей и партнёров
